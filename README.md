# Java GUI Calculator using Swing

This project is a simple Calculator Application developed using Java Swing and Event Handling. The calculator provides a graphical user interface (GUI) where users can perform basic arithmetic operations such as Addition, Subtraction, Multiplication, and Division.

**Features**

User-friendly graphical interface
Perform basic arithmetic operations:

Addition

Subtraction

Multiplication

Division

Input fields for entering numbers

Result display field

Error handling for invalid inputs

Event-driven programming using ActionListener

**Technologies Used**

Java

Java Swing

AWT Event Handling

**Concepts Used**

JFrame for the main window

JTextField for user input and result display

JButton for operation buttons

ActionListener for button click events

Exception handling using try-catch

# Java Quiz Application

A simple Console-Based Quiz Application developed using Java and Object-Oriented Programming (OOP) concepts. This project allows players to answer multiple-choice questions, calculates scores, and displays the final result at the end of the quiz.

**Features**

Multiple-choice quiz system

Player name input

Score tracking system

Displays correct and wrong answers

Final result summary

Object-Oriented Programming implementation

Beginner-friendly Java project

**Technologies Used**

Java

OOP Concepts

Arrays

Scanner Class

Conditional Statements

Loops

**OOP Concepts Implemented**

1) **Encapsulation**

Private data members

Getter methods for accessing data

2) **Classes and Objects**

Question class for storing quiz questions

Player class for managing player details and score

**Constructors**

Used to initialize question data and player information

Project Structure

**Question Class**

Handles:
Question text

Options array

Correct answer

**Player Class**

Handles:

Player name

Score management

Score increment

**Main Class**

Handles:

Quiz flow

User input

Answer checking

Final result display


**Conclusion**

This Java Quiz Application is a great beginner project for understanding Java programming and Object-Oriented Programming concepts. It provides hands-on practice with classes, objects, arrays, loops, and user interaction through the console.
